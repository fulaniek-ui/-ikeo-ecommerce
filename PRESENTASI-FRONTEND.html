Latar Belakang

Motivasi mengikuti bootcamp adalah untuk mendalami dunia Frontend dan Backend development guna membangun aplikasi web end-to-end yang modern dan terukur.

Lingkup Proyek

Project-Ikeo: Aplikasi E-Commerce interaktif pelanggan.

Final-Project-Bootcamp: Sistem Admin Backend terpadu.

🚀 Tech Stack

⚛️
React.js Frontend

Membangun antarmuka yang sangat interaktif dan responsif (SPA) dengan pengelolaan state yang efisien untuk aplikasi toko online.

🐘
Laravel Backend

Arsitektur MVC yang tangguh untuk memproses logika bisnis, sistem otentikasi, manajemen API, dan database relasional.

Pembelajaran & Tantangan

💡

💡 Hal yang Dipelajari

  • • Memahami arsitektur integrasi Frontend (React) dan Backend API (Laravel) secara mendalam.
  • • Praktik terbaik desain struktur Database Relasional untuk sistem E-Commerce yang kompleks.
  • • Manajemen state global dan lokal yang efisien untuk optimasi performa UI di React.
🚧

🚧 Tantangan Utama

  • • Isu keamanan CORS saat pertama kali menghubungkan dua environment lokal yang berbeda.
  • • Sinkronisasi data *Cart* (Keranjang) antara memori browser pengguna dan database server secara real-time.
  • • Mendesain antarmuka (UI/UX) yang intuitif dan profesional dalam tenggat waktu yang ketat.
